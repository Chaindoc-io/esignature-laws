# esignature-laws

Open dataset and guides on electronic signature laws by jurisdiction. Site: [laws.chaindoc.io](https://laws.chaindoc.io).

## What's in here

- `docs/<region>/<country>.md` — narrative guides (≥800 words each), grounded in primary sources.
- `data/laws.json` — machine-readable database of the same facts, validated by `data/schema.json`.
- `index.md` — landing page for the published site.

## Validation

Every fact in this repo traces to a primary source (statute, regulation, official court ruling). See `CONTRIBUTING.md` for the contribution process.
